## 共同变量说明

| 变量名 | 公式 | 说明 |
|--------|------|------|
| `P` | 播放量 | 视频累计播放数 |
| `L` | 点赞数 | - |
| `C` | 投币数 | 权重 ×2 |
| `F` | 收藏数 | 权重 ×1.5 |
| `M` | 评论数 | 权重 ×1.5 |
| `D` | 弹幕数 | 权重 ×0.5 |
| `S` | 分享数 | 权重 ×1 |
| `T` | 发布后天数 | 当前日期 - 发布日期 |
